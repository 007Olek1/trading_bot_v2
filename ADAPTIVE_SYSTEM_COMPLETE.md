## 🎯 ПАРАМЕТРЫ ПО РЕЖИМАМ

### 🟢 БЫЧИЙ РЫНОК (BULL)
**Стратегия**: Агрессивная торговля
```
- Уверенность: 70% (можно рисковать)
- Цель прибыли: +15% (высокие цели)
- Trailing SL: активация после +2.0%
- MTF выравнивание: 3/5 (мягче)
- Макс. позиций: 3
```

### 🔴 МЕДВЕЖИЙ РЫНОК (BEAR)
**Стратегия**: Консервативная торговля
```
- Уверенность: 82% (очень строго!)
- Цель прибыли: +8% (быстрые цели)
- Trailing SL: активация после +1.5%
- MTF выравнивание: 5/5 (все таймфреймы!)
- Макс. позиций: 1 (минимум риска)
```

### 🟡 БОКОВОЙ РЫНОК (SIDEWAYS)
**Стратегия**: Сбалансированная торговля
```
- Уверенность: 75%
- Цель прибыли: +12%
- Trailing SL: активация после +2.5%
- MTF выравнивание: 4/5
- Макс. позиций: 2
```

### ⚡ ВОЛАТИЛЬНЫЙ РЫНОК (VOLATILE) ← ТЕКУЩИЙ
**Стратегия**: Осторожная торговля
```
- Уверенность: 80% (высокая уверенность)
- Цель прибыли: +10% (средние цели)
- Trailing SL: активация после +1.8% (быстро защищаем)
- MTF выравнивание: 4/5
- Макс. позиций: 2
```
