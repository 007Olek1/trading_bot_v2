## 🔢 ШАГ 3: РАСЧЁТ ИНДИКАТОРОВ

### На основном таймфрейме (15m) рассчитываются:

#### 1. EMA (Exponential Moving Average):
```
EMA 20:  $0.494212
EMA 50:  $0.492336
EMA 200: $0.493871
Цена:    $0.495800

Анализ: Цена > EMA20 > EMA50 → ВОСХОДЯЩИЙ ТРЕНД
```

#### 2. RSI (Relative Strength Index):
```
RSI: 49.6

Зоны:
  > 70  → Перекупленность (сигнал SHORT)
  < 30  → Перепроданность (сигнал LONG)
  30-70 → Нейтральная зона ✅
```

#### 3. MACD (Moving Average Convergence Divergence):
```
MACD Line: Быстрая EMA - Медленная EMA
Signal Line: EMA от MACD
Histogram: MACD - Signal

Сигналы:
  MACD пересекает Signal снизу вверх → LONG
  MACD пересекает Signal сверху вниз → SHORT
```

#### 4. ADX (Average Directional Index):
```
ADX: Сила тренда

Значения:
  < 25  → Слабый тренд / боковик
  25-50 → Умеренный тренд
  > 50  → Сильный тренд
```

#### 5. Bollinger Bands:
```
Upper Band: SMA + (2 × StdDev)
Middle Band: SMA(20)
Lower Band: SMA - (2 × StdDev)

Сигналы:
  Цена касается нижней полосы → возможен отскок (LONG)
  Цена касается верхней полосы → возможен откат (SHORT)
```

#### 6. Объём:
```
Текущий объём: 63,167
Средний объём: 1,312,878
Отношение: 0.05x

Анализ:
  > 1.5x → Всплеск объёма (сильный сигнал)
  < 1.5x → Нормальный объём
```

## 📭 ПОЧЕМУ БОТ НЕ НАХОДИТ СИГНАЛЫ?

### Из логов:

### Возможные причины:

1. **Низкая уверенность (<65%)**
   - Стратегии дают слабые сигналы
   - Рынок в неопределённости

2. **Недостаточное MTF выравнивание (<3)**
   - Таймфреймы не согласованы
   - Разнонаправленные тренды

3. **Низкий объём (<1.3x)**
   - Нет активности на рынке
   - Слабый интерес трейдеров

4. **Расхождение стратегий**
   - Одна говорит LONG, другая SHORT
   - Сигнал отклоняется

5. **Лимит позиций достигнут**
   - Уже открыто 3 позиции
   - Ждём закрытия
