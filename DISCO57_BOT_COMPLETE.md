# ✅ БОТ DISCO57 СОЗДАН

## Что такое Disco57?

**Disco57** - это адаптивное правило обучения (DiscoRL), а НЕ название бота.

Оно используется для:
- Адаптации весов ансамбля моделей
- Самообучения на основе результатов сделок
- Оптимизации стратегии в реальном времени

## Ключевые особенности

### 1. Правильная торговля
- ✅ Только BUY и SELL сигналы (HOLD для пропуска)
- ✅ Максимум 3 позиции одновременно
- ✅ Без усреднения и повторных входов
- ✅ Автоматический SL/TP на каждую позицию

### 2. Мультифреймовый анализ
- ✅ 30m → 1h → 4h → 1d
- ✅ Минимум 2 таймфрейма должны подтверждать сигнал
- ✅ Уверенность ≥70% для входа

### 3. Технические индикаторы
- **Тренд:** EMA 20/50/200, ADX
- **Импульс:** RSI, Stochastic RSI, MACD
- **Волатильность:** Bollinger Bands, ATR
- **Объём:** OBV, Volume Ratio

### 4. Риск-менеджмент
- **SL:** -20% от позиции (например, $10 → SL $2)
- **TP:** +10% от позиции (например, $10 → TP $1)
- **Trailing TP:** Активируется при +5% прибыли
- **Автозакрытие:** Через 24 часа

### 5. Режимы рынка
- **Trending:** Активная торговля (ADX > 25)
- **Ranging:** Торговля приостановлена (флэт)
- **Volatile:** Торговля с осторожностью

## Запуск на сервере

```bash

scp -r bybit_futures_bot root@185.70.199.244:/root/

# 2. Установка зависимостей
ssh root@185.70.199.244
cd /root/bybit_futures_bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 3. Запуск
nohup python main.py > /tmp/bot.log 2>&1 &

# 4. Мониторинг
tail -f logs/bot.log
```
