### Предложение 3: Частичное закрытие позиций (ПРИОРИТЕТ 1)

**Что добавить:**
Закрывать позицию частями на разных TP уровнях

**Текущий подход:**
```
TP: +12% → закрыть 100% позиции
```

**Новый подход:**
```
TP1: +2% → закрыть 30% позиции (зафиксировать прибыль)
TP2: +4% → закрыть 30% позиции
TP3: +6% → закрыть 20% позиции
TP4: +12% → закрыть остаток 20%
```

**Преимущества:**
- ✅ Фиксирует прибыль раньше
- ✅ Снижает риск отката
- ✅ Оставляет позицию для больших движений
- ✅ Увеличивает Profit Factor

**Код:**
```python
async def partial_close_position(self, symbol: str, close_percent: int):
    """Частичное закрытие позиции"""
    pos = self.open_positions[symbol]
    close_qty = pos['quantity'] * (close_percent / 100)
    
    # Закрыть часть
    side = "Sell" if pos['direction'] == "LONG" else "Buy"
    order = self.client.place_order(
        category="linear",
        symbol=symbol,
        side=side,
        orderType="Market",
        qty=str(close_qty),
        reduceOnly=True
    )
    
    pos['quantity'] -= close_qty
```

```python
async def check_position(self, symbol: str):
    # ... существующий код ...
    
    # Добавить проверку частичного закрытия
    if config.USE_PARTIAL_CLOSE:
        await self.check_partial_close(symbol, current_price)

async def check_partial_close(self, symbol: str, current_price: float):
    """Частичное закрытие на TP уровнях"""
    pos = self.open_positions[symbol]
    
    # Рассчитываем PnL
    if pos['direction'] == 'LONG':
        pnl_percent = ((current_price - pos['entry_price']) / pos['entry_price']) * 100
    else:
        pnl_percent = ((pos['entry_price'] - current_price) / pos['entry_price']) * 100
    
    # Проверяем уровни
    for level in config.PARTIAL_CLOSE_LEVELS:
        level_key = f"partial_{level['profit_percent']}"
        
        if level_key in pos.get('partial_closes', []):
            continue
        
        if pnl_percent >= level['profit_percent']:
            # Закрываем часть
            close_qty = pos['quantity'] * (level['close_percent'] / 100)
            await self.partial_close(symbol, close_qty)
            
            if 'partial_closes' not in pos:
                pos['partial_closes'] = []
            pos['partial_closes'].append(level_key)
```
