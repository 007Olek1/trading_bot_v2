### 6. 💰 УПРАВЛЕНИЕ КАПИТАЛОМ

#### A. Kelly Criterion
```python
def calculate_kelly_position_size(win_rate, avg_win, avg_loss):
    """Оптимальный размер позиции по Kelly"""
    kelly = (win_rate * avg_win - (1 - win_rate) * avg_loss) / avg_win
    return kelly * 0.5  # Half Kelly для безопасности
```
