## 🔧 ИСПРАВЛЕННЫЕ ПРОБЛЕМЫ:

1. ✅ **KeyError: 'confidence'** в команде `/positions`
   - **Причина:** Восстановленные позиции не имели поля `confidence`
   - **Решение:** Добавлена проверка наличия поля перед выводом
   - **Код:**
   ```python
   if 'confidence' in pos:
       msg_parts.append(f"🎯 Уверенность: {pos['confidence']:.1%}\n")
   ```

2. ✅ **Флаг восстановленной позиции**
   - Добавлен вывод флага для восстановленных позиций
   - **Код:**
   ```python
   if pos.get('restored'):
       msg_parts.append(f"🔄 Восстановлена при старте\n")
   ```
